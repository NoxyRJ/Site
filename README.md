# Noxy7 Website

A modern personal portfolio and social hub designed to centralize online presence, gaming platforms, social media profiles, and personalized content in a single, elegant interface.

## Overview

This project was developed to provide a clean, responsive, and visually immersive experience, featuring real-time integrations, multimedia content, and a modern dark-themed design.

## Features

- Personalized profile presentation
- Social media and gaming platform integration
- Discord profile and activity display
- Spotify music integration
- Custom playlists
- Visitor counter
- Responsive layout
- Animated visual effects
- Optimized user interface

## Technologies

- HTML5
- CSS3
- JavaScript

## Purpose

The goal of this project is to create a centralized digital identity platform where visitors can easily access personal information, social networks, favorite content, and online communities.
